Tap a face to select it. Only faces are selectable (no edges/verts). Face edits move its 4 vertices. The canvas renders in the right-hand viewport; watch the centered status text there for loading or error messages.

Face Tools

Subdivide replaces the selected face with 4 smaller faces (2×2). Any stickers on the original face are removed.
Face Color (per face)
Useful for roofs (no PNG) or matching sticker vibes.

Face Transform

Scale is in face-local axes (u/v). Rotate is face-local (x=u, y=v, z=normal). Translate is world XYZ.

Stickers (PNG planes)

Default mode: sticker is a plane that can extend beyond the face edges.
Turn on Clip sticker to face to crop it exactly to the face.
Sticker Transform (applies to selected sticker)
Translate U/V units are in “face sizes”: 1.0 shifts one full face width/height.
In clipped mode, transforms affect the sticker’s UV mapping on the face.

Export

Export includes base mesh + sticker meshes. Clipped stickers export as face-sized quads (no overhang).
Quick “tower frustum” recipe: select top(+y) face → scale 0.8/0.8 → add 4 side stickers + 1 top sticker → set base faces fully transparent → export.
