HUD Arch Buttons

This page demonstrates the minimal standalone HUD arch layout. Update docs/js/hud-arch-config.js to change button ordering, arc length percentages, gaps, and sprite images. Hook your existing ability logic where noted in docs/js/hud-arch.js.

  • Resize-aware: buttons rebuild on window resize.
  • Debug overlay: enable or disable via config.
  • Sprites optional: swap in your own assets.
